About

This sourcecode of this game can be found on Github. de.htwg.se.SetGame.wui is an extension for the de.htwg.se.SetGame project. de.htwg.se.SetGame project has only a graphical user interface (GUI) and a text-based user interface (TUI). This project extends de.htwg.se.SetGame with an web-based user interface (WUI). The main goal of this project is to deepen the acquired knowledge about web-technologies in the study of applied computer science.
